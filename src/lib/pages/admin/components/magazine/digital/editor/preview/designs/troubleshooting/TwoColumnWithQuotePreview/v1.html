Implementation Notes for TwoColumnWithQuotePreview.tsx

Key Changes Needed:

  1. Replace CSS columns with flexbox two-column layout
  2. Split content into left and right column paragraphs (not by position %)
  3. Position quote absolutely in center between columns
  4. Add invisible spacer divs that float to push text around quote
  5. Calculate spacer position based on quotePosition percentage

React Component Structure:

{/* Container with position: relative */}
<div className="two-column-wrapper" style={{ position: 'relative', display: 'flex', gap: '2rem' }}>

  {/* Absolutely positioned quote */}
  <div style={{
    position: 'absolute',
    left: '50%',
    top: `${quotePosition}%`,
    transform: 'translateX(-50%)',
    width: '50%',
    zIndex: 10
  }}>
    {/* Quote content */}
  </div>

  {/* Left Column */}
  <div className="column" style={{ flex: 1 }}>
    {leftParagraphs.map((p, i) => (
      <React.Fragment key={i}>
        {i === spacerIndex && (
          <div style={{ float: 'right', width: '30%', height: '130px' }} />
        )}
        <p>{p}</p>
      </React.Fragment>
    ))}
  </div>

  {/* Right Column */}
  <div className="column" style={{ flex: 1 }}>
    {rightParagraphs.map((p, i) => (
      <React.Fragment key={i}>
        {i === spacerIndex && (
          <div style={{ float: 'left', width: '30%', height: '130px' }} />
        )}
        <p>{p}</p>
      </React.Fragment>
    ))}
  </div>

</div>

Content Splitting Logic:

// Split paragraphs alternately between columns
const paragraphs = articleContent.split('</p>').filter(p => p.trim());
const leftParagraphs = paragraphs.filter((_, i) => i % 2 === 0);
const rightParagraphs = paragraphs.filter((_, i) => i % 2 === 1);

// Calculate spacer index based on quotePosition
const spacerIndex = Math.floor((quotePosition / 100) * leftParagraphs.length);
